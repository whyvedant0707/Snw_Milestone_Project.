# Snw_Milestone_Project.
This is a web application that allows users to explore movies using real-time data from a public API.

CineScope is a responsive web application that allows users to explore movies using real-time data from a public API. Users can search for movies, filter them by criteria such as year or type, sort them based on ratings or release date, and save their favorite movies. The application focuses on providing a smooth and interactive user experience using modern JavaScript features.
