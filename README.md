# 🌟 NetGlass – Smart Movie Explorer

## Live Demo
[Click here to view project](https://whyvedant0707.github.io/Snw_Milestone_Project./)

## Project Description
NetGlass is a web application that allows users to explore movies using real-time data from the OMDb API. The app provides features like search, filtering, sorting, and favorites to enhance user experience.

## Features
-  Search movies by title
-  Filter movies by year or type
-  Sort movies by rating or release date
-  Add/remove favorites (stored in localStorage)
-  Dark mode / Light mode toggle
-  Fully responsive design
