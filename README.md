# 🎬 CineScope – Smart Movie Explorer

## 📌 Project Description
CineScope is a web application that allows users to explore movies using real-time data from the OMDb API. The app provides features like search, filtering, sorting, and favorites to enhance user experience.

## 🚀 Features
- 🔍 Search movies by title
- 🎯 Filter movies by year or type
- 🔄 Sort movies by rating or release date
- ❤️ Add/remove favorites (stored in localStorage)
- 🌙 Dark mode / Light mode toggle
- 📱 Fully responsive design

## 🔌 API Used
OMDb API  
https://www.omdbapi.com/

## 🛠️ Technologies Used
- HTML
- CSS (Flexbox / Grid / Tailwind optional)
- JavaScript (ES6+)
- Fetch API

## 📂 Project Setup
1. Clone the repository
2. Open index.html in browser
3. Add your API key in script.js
